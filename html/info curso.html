Este libro es para encendernos en la noche mas oscura, para atravesar las crisis, para despedirnos de ese amor que acabó, o de ese amor que se fue al cielo. Un libro para reencontramos, para convertirnos en desconocidos que se vuelven amigos, y tienen derecho a llorar, a reir, a tocar fondo y coger impulso hasta tocar las nubes.
